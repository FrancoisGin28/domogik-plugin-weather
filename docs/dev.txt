A specific xPL schema has been defined for the forecast: 

* **xpl-stat** ::

    xpl-stat
    {
    ...
    }
    weather.forecast
    {
        provider=weather.com
        day=<number of the day : 0....>
        day_name=...
        temperature_high=...
